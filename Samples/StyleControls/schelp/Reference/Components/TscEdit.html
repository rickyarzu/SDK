TscEdit

Unit

scControls.pas

TscEdit = class(TCustomEdit)

Description

Edit control with many style kinds and transparency

Properties

 

Name Type Description
LeftButton TscEditButton Adjust left button
RightButton TscEditButton Adjust right button
Transparent Boolean If True then client part of edit is transparent
BorderKind TscEditBorderKind Kind of border...

TscEditBorderKind = (scebFrame, scebBottomLine, scebBottomActiveLine);
BottomLineColor TColor Color for bottom line
ButtonImages TCustomImageList Images for buttons
CustomImages TscImageCollection TscImageCollection component. Bitmap property must be defined in all items, which you will use for custom style
Wallpapers TscImageCollection TscImageCollection component for wallpaper
CustomBackgroundImageNormalIndex
CustomBackgroundImageHotIndex
CustomBackgroundImageDisabledIndex		 Integer Image indexes for states
WallpaperIndex Integer Image index for wallpaper
PromptText String Hint text for edit
HidePromptTextIfFocused Boolean Hide hint text for edit if it focused
UseFontColorToStyleColor Boolean If True then color of the text will be GetStyleColor(Font.Color)
HintComponent TscHint TscHint component to show hints from edit buttons
ContentMarginTop,ContentMarginLeft,
ContentMarginRight, ContentMarginBottom 		Integer Margins for edit content if AutoSize is False
CustomDraw Boolean Set it True to prevent drawing any default content in non-focused control with assigned OnDrawBackground event 

 

Events

 

Name Type Description
OnLeftButtonClick TNotifyEvent Click event for left button
OnRightButtonClick TNotifyEvent Click event for right button
OnDrawBackground TscPaintControlEvent procedure(ACanvas: TCanvas; ARect: TRect;
      AState: TscsCtrlState) of object;

Note, that ARect - is a client (text) area of the editor


Note:

    Use TControl.StyleElements property to adjust custom colors for client and font in all controls when VCL Style is enabled!
