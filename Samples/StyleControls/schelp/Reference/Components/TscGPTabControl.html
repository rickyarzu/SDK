TscGPTabControl

Unit

scGPPagers.pas

Description

Tab control  with vector shapes for tabs, with left, right offset for tabs and advanced features

Properties

  

Name Type Description
Tabs TscAdvancedPagerTabs Collection of TscAdvancedPagerTab
TabOptions TscGPTabOptions
 Property to adjust tab shape, border, text for different states
TabGlowEffect TscButtonGlowEffect Adjust gllow effect for Tab
TabHeight Integer Height of Tab
TabImages TCustomImageList Images for Tabs
Font TFont Font for Tabs
TabIndex Integer Index of the tab
TabsLeftOffset Integer Offset for tabs from left. Use this to put any control on left area before tabs
TabsRightOffset Integer Offset for tabs from right. Use this to put any control on right area after tabs
BorderStyle TscGPTabControlBorderStyle TscGPTabControlBorderStyle = (scgpbsFrame, scgpbsLine, scgpbsNone);
Border style for control
ShowCloseButtons Boolean Show close buttons on tabs
FreeOnClose Boolean If True then Tab and Page will be destroyed when you close page, else Tab and Page will be unvisible only.
MouseWheelSupport Boolean If True then you can change active tab by mouse wheel
ShowInactiveTab Boolean If False then inactive tab will be transparent, else it has a tab-style.
ScrollButtonWidth Integer Width of the scroll button
FrameWidth Integer Width of the frame
FrameScaleWidth Boolean If True then frame of the control and frame for tab will be scaled in Digh-DPI systems
FrameColor TColor Color of the frame
FrameColorAlpha Byte Alpha value for frame color
DrawTextMode TscDrawTextMode TscDrawTextMode = (scdtmGDI, scdtmGDIPlus);

scdtmGDI - draw text with GDI methods
scdtmGDIPlus - draw text with GDI+ methods

Use scdtmGDIPlus value on the Fluent UI background (see TscStyledForm component)

Events

  

Name Type Description
OnChangePage TNotifyEvent Event when page changed
OnClose TscTabCloseEvent TscTabCloseEvent = procedure(Sender: TObject; var CanClose: Boolean) of object;
Set CanClose = True to close page.
OnGetTabDrawParams TscGetAdvTabDrawParamsEvent TscGetAdvTabDrawParamsEvent = procedure(ATabIndex: Integer; ATabState: TscsCtrlState;
    ACanvas: TCanvas) of object;

Use this event to customize tab font (ACanvas.Font), for example.
