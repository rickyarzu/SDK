TscGPGlyphButton

Unit

scGPControls.pas

Description

Button control with with vector shape and vector glyph

Properties

 

Name Type Description
Options TscGPButtonOptions  Property to adjust button shape, border for different states.
Also use StyleColors (default value is True) property to adjust colors from styles(for example, if you set some color to clBtnFace or clHighLight then button will get this color from style (clNone of color property is filling empty).

Properties to adjust kind of the shape:

 - ShapeStyle (scgpssRect, scgpssRoundedRect, scgpRoundedLeftRight,
    scgpSegmentedLeft, scgpSegmentedLeftRounded,
    scgpSegmentedRight, scgpSegmentedRightRounded,
    scgpSegmentedMiddle,
    scgpTabLeft, scgpTabLeftRounded, scgpTabRight,  scgpTabRightRounded, scgpTabTop, scgpTabBottom,
    scgpEllipse, scgpRounded)
  - ShapeFillStyle (scgpsfColor, scgpsfGradient)
  - ShapeFillGradientAngle, ShapeFillGradientPressedAngle
  - ShapeCornerRadius
  - PressedHotColors (set it True if you want that button in pressed state can has hot colors if mouse over)
GlyphOptions TscGPGlyphOptions  Property to adjust vector glyph for button.
Set colors and alpha values  for different states.
Set Thickness of glyph lines.

Set Kind: TscGPButtonGlyphKind property (more than 100 variants of vector glyph (Ok, Cancel, File Open, File Save, Print, Refresh, Gear, Cut, Paste and many more)).

if you set Kind = scgpbgkColorMarker then
  Glypth will looks as color marker to show color value from ColorValue property.

Use Size property to set custom size for image!
Use ThicknessScaled property to scale thickness of image!
ColorValue TColor Value for GlyphOptions.Kind = scgpbgkColorMarker
TextMargin Integer if > -1 then text will be near glyph (not in center) with this offset
Animation Boolean Enabled animation on button
CanFocused Boolean Enabled focus state for button
RepeatClick Boolean Enabled repeat of OnClick event until buttons is pressed
RepeatClickInterval Integer Interval for timer to repeat OnClick event
DropDownMenu TPopupMenu Drop down menu
ShowFocuRect Boolean If True then button draw focus rect
Down Pressed Set pressed state for button
GroupIndex Integer Index of button's group 
AllowAllUp Boolean Specifies whether all buttons in the group that contains this  button can be unselected at the same time
GalleryMenu TscGalleryMenu TscGalleryMenu component
TransparentBackground Boolean if True then button has transparent background
ShowMenuArrow Boolean if GalleryMenu <> nil or DropDownMenu <> nil then arrow will be visible on the button
ShowCaption Boolean Show caption in button
WidthWithCaption,
WidthWithoutCaption		 Integer Use these values of button width when you set ShowCaption property.
ArrowPosition TscArrowPosition TscArrowPosition = (scapRight, scapBottom);

Positiion of the arrow in button
ArrowDirection TscArrowDirection TscArrowPosition = (scadDefault, scadRight);

Direction of the arrow in button.
If Value is scadRight then drop down menu position will be from right also.
WordWrap Boolean If True then caption of button will be with word wrap.
CustomDropDown Boolean If True then button will work with dropdown form.

See also TscStyledForm
ScaleFrameWidth Boolean If True then control will scale frame width (depends on the system DPI)
Badge TscGPButtonBage Options for button badge
DrawTextMode TscDrawTextMode TscDrawTextMode = (scdtmGDI, scdtmGDIPlus);

scdtmGDI - draw text with GDI methods
scdtmGDIPlus - draw text with GDI+ methods

Use scdtmGDIPlus value on the Fluent UI background (see TscStyledForm component)
FluentLightEffect Boolean Enable fluent light effect - light with mouse moving
ToggleMode Boolean If True then button can toggle down state (checkbox behavior). Check Down property in OnClick event.

CloseUp Use this method on dropdown form to redraw button after closing of dropdown form...

procedure TDropDownForm.scStyledForm1CloseUp(ADropDownForm: TCustomForm;
  ADropDownControl: TControl; AAcceptChanges: Boolean);
begin
  if ADropDownControl <> nil then
  begin
    if ADropDownControl is TscButton then
      TscButton(ADropDownControl).CloseUp(AAcceptChanges);
  end;
end;
