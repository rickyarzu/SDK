TscRadioGroup

Unit

scControls.pas

Description

RadioGroup control, which baced from TscGroupBox and TscRadioButton controls
This control has same propeties as TRadioGroup control + additional...

Properties

 

Name Type Description
ButtonsImages TCustomImageList Images for buttons
ButtonsAnimation Boolean Enable animation on button
ButtonsGlowEffect TButtonGlowEffect Adjust glow effect for button
ShowFocuRect Boolean If True then buttons draw focus rect

Events

 

Name Type Description
OnButtonClick TNotifyEvent Event on button's click

 

Note:

    Use TControl.StyleElements property to adjust custom colors for client and font in all controls when VCL Style is enabled!
