TscGPVirtualImageList

Unit

scGPImages.pas

Description

TscGPVirtualImageList - image list component, which can uses one external TImageList as source for images.
You can set dynamically size, scale factor (High-DPI support is easy now) for images and get smooth HQ scaling!

Properties

Name Type Description
Width Integer Virtual width of the images
Height Integer Virtual height of the images
DefaultWidth Integer Default virtual width of the images. SetScaleFactor method will uses this value to calc actual size with scale factor
DefaultHeight Integer Default virtual height of the images. SetScaleFactor method will uses this value to calc actual size with scale factor
SourceImageList TImageList TImageList component with source images. ColorDepth property in it must be cd32Bit!
You can use one TImageList for different TscGPVirtualImageList components.

You can dynamicly set Width and Height properties as you want.
SourceImageCollectionTscGPImageCollectionTscGPImageCollection component with any images

Use it as source for images. TscGPVirtualImageList will automatically drawing scaled images.
InterpolationModeTscGPInterpolationModeTscGPInterpolationMode =(scgppimDefault, scgppimHighQualityBilinear);

scgppimDefault - high quality interpolation by default
scgppimHighQualityBilinear - high quality bilinear inerpolation
DirectDrawBooleanIf True (default value) then component scales and draws images directly from sources.
Can be used with controls, which uses TCustomImageList.Draw method.
If False then component generate scaled internal images.
Can be used with control, which uses system ImageList_Draw, ImageList_DrawEx methods (TToolBar, TTreeView, TListView)



Methods

 

Name Description
SetScaleFactor procedure SetScaleFactor(Value: Double);

Use this method in scaled form and in applications with High-DPI option.
It automatically set virtual sizes for images.
For example, you can use TscStyledForm.OnChangeScale event:

procedure TForm1.scStyledForm1ChangeScale(AScaleFactor: Double);
begin
  scGPVirtualImageList1.SetScaleFactor(AScaleFactor);
  scGPVirtualImageList2.SetScaleFactor(AScaleFactor);
end;
UpdateUse this method to synchronize count value in TscGPVirtualImageList and in SourceImageList or SourceImageCollection
