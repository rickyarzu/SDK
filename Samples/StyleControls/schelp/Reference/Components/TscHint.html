TscHint

Unit

scHints.pas

Description

Component to show custom hints. Hints can be with title, image and text.

Properties


Name Type Description
LineSeparator String Separator for lines. Default values is "@@".
If you want to show multiline hint then add LineSeparator to the end of your line.
For example: "Line1@@Line2"
AlphaBlend Boolean If True the hints with be with alphablend effect
AlphaBlendValueByteAlphaBlend value
AlphaBlendAnimationBooleanEnable / Disable showing animation for hint
ActiveForAllApplicationBooleanSet it to True and replace all hints in applications to hints from TscHint component (this helps you to fix all problems with hints on multi-monitors High-DPI systems)
