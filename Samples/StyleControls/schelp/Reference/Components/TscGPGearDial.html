TscGPGearDial

Unit

scGPMeters.pas

Description

Rounded dial control with big rotation angle

Properties

 

Name Type Description
MinValue Double Min value
MaxValue Double Max value
Value Double Current value
FillColor TColor Fill color for control shape (clNone - fill is missing)
FillColorAlpha Byte Alpha value of the fill color for control shape
FillStyle TscGPShapeFillStyle TscGPShapeFillStyle = (scgpsfColor, scgpsfGradient);

scgpsfColor - solid color
scgpsfGradient - auto gradient from FillColor
FillGradientAngle Integer Angle of fill gradient
FrameColor TColor Color of the frame (clNone - frame is invisible)
FrameWidth Integer Width of the frame
CanFocused Boolean Enable focus on control
KnobColor TColor Color of the knob
KnobColorAlpha Byte Alpha value for knob color
RotationAngle Integer Rotation angle. It can be > 360, for example, 3600 - 10 turns
StartAngle Integer Angle to start rotation
ImageCollection TscCustomImageCollection Image collection component
ImageIndex Integer Index of image from image collection
ShowImage Boolean Show / Hide image
FormatStrValue String Float format to display value
ShowValue Boolean If True then control shows value
DrawTextMode TscDrawTextMode TscDrawTextMode = (scdtmGDI, scdtmGDIPlus);

scdtmGDI - draw text with GDI methods
scdtmGDIPlus - draw text with GDI+ methods

Use scdtmGDIPlus value on the Fluent UI background (see TscStyledForm component)

Events

 

Name Type Description
OnChange TNotifyEvent Event when value changed
OnLastChange TNotifyEvent Event when value changed and you release thumb capture
