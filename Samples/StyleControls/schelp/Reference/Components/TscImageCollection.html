32-bit bitmap drawn much more faster than PNG image. This is why the best for performance convert PNG image to the 32-bit bitmap or 24-bit bitmap

 Boolean
 This values define which part of the bitmap has alpha-channel. This need for optimization of performance. When you set Bitmap from PNG then dialog automatically scan Bitmap and set this properties, but you can set values as you need.
AlphaBlend - alpha-channel of client part of the bitmap
AlphaBlendBorder - alpha-channel of border
AlphaBlendCorners - alpha-channel of corners
