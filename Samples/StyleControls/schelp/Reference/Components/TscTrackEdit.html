TscTrackEdit

Unit

scControls.pas

TscTrackEdit = class(TscEdit)

Description

Edit control to work with Integer values and change it with popup trackbar

Properties


 
Name Type Description
Increment Integer Increment of value when you use up and down keys
MinValue Integer Min value
MaxValue Integer Max value
Value Integer Current value
DblClickShowTrackBaer Boolean if True then you can show popup with dbl click event
SupportUpDownKeys Boolean Enable using up and down keys
JumpWhenClickBooleanEnable JumpWhenClick property of trackbar (set value by click on track area)
TrackBarWidthIntegerWidth of trackbar, if value = 0 then width of trackbar = width of control
TrackBarHeightIntegerHeight of trackbar, if value = 0 then control set height of trackbar automatically 

 

Note:     Use TControl.StyleElements property to adjust custom colors for client and font in all controls when VCL Style is enabled!
