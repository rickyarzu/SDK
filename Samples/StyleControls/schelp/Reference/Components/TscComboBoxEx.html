TscComboBoxEx

Unit

scControls.pas

TscComboBox = class(TCustomComboBoxEx)

Description

ComboBoxEx control with additional features

Properties


Name Type Description
SelectionStyle TscSelectionStyle Kind of selection:

TscSelectionStyle = (scstStyled, scstColor);
scstStyled - draw selection as bitmap from theme or VCL Style
scstColor - draw selection as highlight color from theme or VCL Style
SelectionColor TColor Custom selection color for control (value must be <> clNone)
SelectionTextColor TColor Color of text for custom selection color
StyleKindTscComboStyleKindTscComboStyleKind = (scscbDefault, scscbPushButton);

if value is scscbPushButton then control will use style as push button.

 

Note:

    Use TControl.StyleElements property to adjust custom colors for client and font in all controls when VCL Style is enabled!
