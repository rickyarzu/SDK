TscSplitView

Unit

scModernControls.pas

TscSplitView = class(TscPanel)

Description

Panel, which can be opened and closed. 

Properties

 

Name Type Description
Placement TscSplitViewPlacement TscSplitViewPlacement = (scsvpLeft, scsvpRight, scsvpTop, scsvpBottom);
Position of control on parent control.
DisplayMode TscSplitViewDisplayMode TscSplitViewDisplayMode = (scsvmOverlay, scsvmDocked);

scsvmOverlay - control over other controls on the parent in opened state
scsvmDocked - control offseet other controls on the parent in opened state
CompactWidth Integer Width in closed state. If Width = 0 then control not visible.
(for scsvpLeft, scsvpRight placement)
CompactHeight Integer Height in closed state. If Height = 0 then control not visible.
(scsvpTop, scsvpBottom placement)
OpenedWidth Integer Width in opened state.
(scsvpLeft, scsvpRight placement)
OpenedHeight Integer Height in opened state.
(scsvpTop, scsvpBottom placement)
Opened Boolean If True then control will overlap other controls with Width = OpenedWidth or Height = OpenedHeight
Animation Boolean Enable open / close animation
HideControls Boolean If true then visible controls will be hidden when Opened = False
AnimationStep Integer Value to increase size of the control in animation cycle
GripSizeIntegerif Value > 0 then you can resize control in opened state. GripSize value define  capture area from right or bottom of the control to resize it.
OpenedMinWidth, 
OpenedMaxWidth,
OpenedMinHeight,
OpenedMaxHeight		IntegerValues for size limitations, which controls uses when you change size with GripSize > 0
AnimationTypeTscSplitViewAnimationTypeTscSplitViewAnimationType = (scsvaInertial, scsvaLinear);

scsvaInertial - animation with intertial
scsvaLinear - linear animation with AnimationStep
