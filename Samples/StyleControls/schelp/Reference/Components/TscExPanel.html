TscExPanel

Unit

scExtControls.pas

Description

Panel control, which can roll horizontally or vertically.

Properties

Name Type Description
Caption String Text on control
BackgroundStyle TscExPanelBackgroundStyle TscExPanelBackgroundStyle =
    (scexbgsPanel, scexbgsFormBackground, scexbgsTransparent);

Style for control background.  You can use Color property with scexbgsPanel style.
RollKind TscExPanelRollKind TscExPanelRollKind = (scrkRollHorizontal, scrkRollVertical);

This property define how control will be roll: horizontally or vertically
RollUpState Boolean If True then panel is in rollup state
ShowFrame Boolean Show frame of the panel
RealWidth,
RealHeight		 Integer Width and Height when RollUpState = False.
With RollKind = scrkRollHorizontal panel uses RealWidt value and
with RollKind = scrkRollVertical panel uses RealHeight value. 
ShowRollButton Boolean Show button to change RollUpState
ShowCloseButton Boolean Show button to change Visible of the control to False
CaptionHeight Integer Height of the caption. Also this values defines size of the control when RollUpState = true
CaptionImages TCustomImageList Image list for caption
CaptionImageIndex Integer Image index for caption
Spacing Integer Spacing between image and text
Moveable Boolean If True then you can move panel with caption by mouse (as with form)
Sizeable Boolean if True then you can change panel size with mouse (as with form)
MovingInParentСlientBounds Boolean If True then when you moving control then it can be moved only in parent client area
StorePaintBuffer Boolean If True then transparent child controls from StyleControls VCL will use special paint buffer to get parent image. This solution really increase performance!
HeaderStyleTscExPanelHeaderStyleTscExPanelHeaderStyle =   (scexphsHeader, scexphsColor);
Style for header
HeaderColorTColorColor value if TscExPanelHeaderStyle = scexphsColor
ButtonGlyphColorTColorColor value for button glyph  if TscExPanelHeaderStyle = scexphsColor
ChangeRollStateWithCaptionClickBooleanEnable changing of roll state with caption click
BorderWidthIntegerBorder width, which defines border to resize control (Sizeable = True) and client area
FrameColorTColorColor of the frame
ShowFrameBooleanShow / hide frame
HorzRollButtonPositionTscExPanelHorzRollButtonPositionTscExPanelHorzRollButtonPosition = (scrbpRight, scrbpLeft);

Position of roll button in horizontal mode.

 

Note:

    Use TControl.StyleElements property to adjust custom colors for client and font in all controls when VCL Style is enabled!
