TscGPPageViewer

Unit

scGPPagers.pas

Description

Simple page control  to switch between pages only (analogue of TNoteBook).

It can be fully transparent.

Properties

  

Name Type Description
Pages TscGPPageViewerItems Collection of TscGPPageViewerItem.
Use TscPageViewerItem.Page property to get page for item.
Component use this collection to get index of page. Also you can add, delete or reorder items (pages) with collection editor.
PageIndex Integer Index of page in collection
ActivePage TscGPPageViewerPage TscPageViewerPage = class(TscGPPanel)

Events

  

Name Type Description
OnChangePage TNotifyEvent Event when page was changed
