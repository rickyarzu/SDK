TscPageControl

Unit

scControls.pas

TscPageControl = class(TPageControl)

Description

PageControl with different styles, wallpapers for pages and with additional features for tabsheet (see TscTabSheet)

Properties


Name Type Description
HideTabs Boolean Hide / show all tabs
HideBorders Boolean Hide / show border
MouseWheelSupportBooleanIf True then you can change page with mouse wheel
ShowButtonDividerBooleanIf True then control draw divider between buttons when Style = tsFlatButtons and Multiline = False
ShowCloseButtonBooleanIf True then tab has close button
ShowFocusRectBooleanIf True then control draw focus rect on tab
ShowInActiveTabBooleanIf False then control not draw inactive tab (only image and text)
TabExtendedDrawBooleanIf True then control use extended methods to draw tab's content. You can use TabMargin, TabSpacing, TabLayout, TabGlowEffect properties to adjust tabs.
TabGlowEffectTscButtonGlowEffectGlow effect for text and image 
TabImageGlowBooleanIf True then TabGlowEffect can apply for image in tab
TabLayoutTButtonLayoutLayout of image in tab
TabMarginIntegerMargin of image and tex in tab
TabSpacingIntegerSpacing between image and text in tab
TabsInCenterBooleanIf True then Tabs will draw in center of control. TabWidth must be > 0 in this case. Control emulate this behaviour and in designing time when you click on control it can offset position if you not set Align property.This property work only with TabPosition = tpTop.
TabsOffsetIntegerOffset for tabs from left. This property work only with TabPosition = tpTop. Control emulate this behaviour and in designing time when you click on control it can offset position if you not set Align property. UpDown control is disabled in this case.
WallpapersTscImageCollectionWallpapers for tabs and pages
TabsWallpaperIndexIntegerIndex of wallpapers for tab's background
FreeOnCloseBooleanIf True the control destroy page else TabVisible will be False
