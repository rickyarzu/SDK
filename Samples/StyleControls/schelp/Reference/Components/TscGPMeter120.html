TscGPMeter120

Unit

scGPMeters.pas

Description

Rounded meter control with 120 degrees of scale

Properties

 

Name Type Description
MinValue Double Min value
MaxValue Double Max value
Value Double Current value
FormatStrValue String Float format to display value text
FormatLabelValue String Float format to display tick labels
AutoSizeFont Boolean If True then font height will be automatic else control uses Font.Size property
FrameColor TColor Color of the frame (clNone - frame is invisible)
FrameWidth Integer Width of the frame
CenterFrameColor TColor Frame color of the center circle
CenterFrameColorAlpha Byte Alpha value for frame color of the center circle
CenterFillColor TColor Fill color of the center circle
CenterFillColorAlpha Byte Alpha value for the fill color of the center circle
CenterFillStyle TscGPShapeFillStyle Fill style for center circle
CenterFillGradientAngle Integer Angle of fill gradient for center circle
CenterFrameWidth Integer Frame width of the center circle
ArrowType TscGPMeterArrowType = (scgpatLine, scgpatArrow); Type of the arrow: line or arrow
ArrowColor TColor Arrow color
ArrowColorAlpha Byte Alpha value for arrow color
ArrowWidth Integer Width of the arrow
ValueTextColor TColor Color for value text (ShowValue = True)
ValueHint String Hint for value (%, rpm)
ValueHintColor TColor Color for hint value text
ScaleSteps Integer Count of the scale steps
ScaleSubSteps Integer Count of the scale steps between big ticks (define small ticks)
ScaleSections TscScaleSections Collection of sections on scale arc. You can set color, alpha value, StartValue and EndValue for each section
ScaleDivider Double Divider value of tick numbers (If Value = 3000 and Divider = 1000 then you get tick with "3" label)
TicksColor TColor Color of the scale ticks
TicksColorAlpha Byte Alpha value for color of the scale ticks
TicksWidth Integer Width of the tick on scale
TicksSmallWidth Integer Width for small ticks
ShowValue Boolean if True then control shows current value on the bottom
ShowScaleTicks Boolean If True then control shows ticks on the scale
ShowScaleLabels Boolean If True then control shows labels on the scale
DrawTextMode TscDrawTextMode TscDrawTextMode = (scdtmGDI, scdtmGDIPlus);

scdtmGDI - draw text with GDI methods
scdtmGDIPlus - draw text with GDI+ methods

Use scdtmGDIPlus value on the Fluent UI background (see TscStyledForm component)

Events

 

Name Type Description
OnGetArrowColor TscGetColorEvent procedure(Sender: TObject; var AColor: TColor; var AColorAlpha) of object;

Use this event to set custom arrow color depending on the value
OnGetCenterColor TscGetColorEvent Use this event to set custom center color depending on the value
OnGetCenterFrameColor TscGetColorEvent Use this event to set custom center frame depending on the value
