Description

Switch control with vector graphic and Windows 10 UWP design

Name Type Description
State TscSwitchState TscSwitchState = (scswOff, scswOn);
State of switch
StyleKind TscSwitchStyleKind TscSwitchStyleKind = (scswsStyled, scswsColor, scswsCustomImage);
Style kind of switch:

 * scswsStyled - control uses elements from style (since RAD Studio 10 Seattle) and works in scswsColor mode in other versions
 * scswsColor - control uses colors for frame and thumb, which has style as in Windows Modern Application
 *  scswsCustomImage - control uses custom images for frame and thumb

FrameSolidBooleanIf True then frame is solid in Off state
FrameOnSolidBooleanIf True then frame is solid in On state
