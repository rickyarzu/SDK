scGPExtControls.pas

Description

Panel control with scroll feature

Properties


Name Type Description
Wallpapers TscImageCollection Picture for wallpaper
WallpaperIndex Integer Index of image
CustomImages TscImageCollection Image for additional background
CustomImageIndex Integer Index of image for additional background
StyleKind TscGPScrollPanelBGStyleKind Style of panel:

 TscGPScrollPanelBGStyleKind = (scgspbFormBackground, scgspbPanel,
    scgspbTransparent);
Color TColor Color of panel for scsPanel style
HotScroll Boolean If True then scrolling start when mouse over scroll button
ScrollOffset Integer Scroll offset value. If Value = 0 then scroll offset is client size.
ScrollTimerInterval Integer Scroll timer interval
ScrollType TscScrollType Scroll direction:

TscScrollType = (scstHorizontal, scstVertical);
TouchScroll Boolean Enable scrolling by touch events
ScrollButtonWidth Integer Size of the scroll button
FillColor TColor Color for background
FillColor2 TColor Second color for background with gradient style.
If value is clNone then this color will be ignored and gradient filling will uses only one color
FillColorAlpha Byte Alpha value for color of background
FillStyle TscGPShapeFillStyle TscGPShapeFillStyle = (scgpsfColor, scgpsfGradient);

Filling style for panel.
 scgpsfColor - one color
 scgpsfGradient - control automatically create light and dark colors for gradient from FillColor value.
FillGradientAngle Integer Angle for gradient (90, -90, 45, -125, ....)
FillGradientBeginAlpha Byte Alpha value for color in the begin of gradient
FillGradientEndAlpha Byte Alpha value for color in the end of gradient
FillGradientBeginColorOffset Byte Lightness offset in percents for first color in gradient
FillGradientEndColorOffset Byte Darkness offset in percents for second color in gradient
StorePaintBuffer Boolean If True then transparent child controls from StyleControls VCL will use special paint buffer to get parent image. This solution really increase performance!
HorzScrollDirection TscGPScrollPanelHorzDirection TscScrollPanelHorzDirection = (scgspsdLeftToRight, scgspsdRightToLeft);

Direction of horizontal scrolling: left to right (default)  or right to left
VertScrollDirection TscGPScrollPanelVertDirection TscScrollPanelVertDirection = (scgspsdTopToBottom, scgspsdBottomToTop);

Direction of vertical scrolling: top to bottom default) or bottom to top
