TscLinkBar

Unit

scAdvancedControls.pas

Description

Analogue of listbox control, but with links as items, which have glow effect and  many more for adjutment

Properties

  

Name Type Description
SelectionGlow TscGlowEffect Adjust gllow effect for selection
Items TscLinkBarItems Collection of TscLinkBarItem
ItemIndex Integer Index of active item
ItemHeight Integer Height of item
HeaderHeight Integer Height of items with Header = True
HeaderFont TFont Font for header items
HeaderStyle TscAdvancedHeaderStyle Style for header items...

TscAdvancedHeaderStyle = (scahsDefault, scahsBottomLine);

scahsDefault - use header theme or style
scahsBottomLine - use bottom line in transparent header
HeaderUseStyleColor Boolean If True then headers will use colors from VCL Styles
BorderStyle TscBorderStyle Border style...

TscBorderStyle = (scbsSingle, scbsNone);
BackgroundStyle TscBackgroundStyle Style for background...

TscBackgroundStyle = (scbgsColor, scbgsFormBackground, scbgsTransparent);
Wallpapers TscImageCollection Wallpapers
WallpaperIndex Integer Index of picture for wallpaper
Color TColor Color value for scbgsColor background style
UseHandPointCursorBooleanUse HandPoint cursor when mouse over item
ShowTextUnderLineBooleanShow text underline when mouse over item
HoldSelectedItemBooleanHold item, which you select by mouse
SpacingIntegerSpacing between image and text in item
Images TCustomImageList Images for items
