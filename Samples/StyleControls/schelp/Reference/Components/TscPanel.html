TscPanel

Unit

scControls.pas

Description

Panel control with a lot of style and border kinds, wallpapers, transparency and many more

Properties

Name Type Description
Wallpapers TscImageCollection Picture for wallpaper
WallpaperIndex Integer Index of picture for wallpaper
CustomImages TscImageCollection Image for additional background
WallpaperIndex Integer Index of image for additional background
StyleKind TscPanelStyleKind Style of panel:

TscPanelStyleKind = (scpsPanel, scpsToolBar, scpsHeader, scpsFormBackground,
    scpsTransparent, scpsEmpty, scpsEdit);

scpsPanel - use color for backhround
scpsToolBar - use toolbar theme and style
scpsHeader - use header theme and style
scpsFormBackground - form client  theme and style
scpsTransparent - control is transparent
scpsEmpty - nothing to draw - you can use this if control is parent for some controls and client part of it not visible
scpsEdit - use edit frame
scpsTabSheet - tabsheet background
BorderStyle TscPanelBorderStyle Style for border: 
(scpbsNone, scpbsFlat, scpbsRaised, scpbsLowered, scpbsLoweredBevel,
    scpbsRaisedBevel,
    scpbsTopBevel, scpbsBottomBevel,
    scpbsTopLightLine, scpbsBottomLightLine,
    scpbsTopShadowLine, scpbsBottomShadowLine,
    scpbsLeftBevel, scpbsRightBevel,
    scpbsLeftLightLine, scpbsRightLightLine,
    scpbsLeftShadowLine, scpbsRightShadowLine,
    scpbsHorzCenterBevel, scpbsVertCenterBevel);

You can use panel as divider also and this property really helps you with that.
Caption String Text on control
ShowCaption Boolean Show text on panel
CaptionGlowEffect TscGlowEffect Adjust glow effect for caption
Alignment TAlignment Alignment for caption
Color TColor Color of panel for scsPanel style
WordWrap Boolean If True then caption will be with word wrap
StorePaintBuffer Boolean If True then transparent child controls from StyleControls VCL will use special paint buffer to get parent image. This solution really increase performance!
FluentUIOpacity Boolean Set it to True to get opacity area of the form with Fluent UI (see TscStyledForm component)

 

Note:

    Use TControl.StyleElements property to adjust custom colors for client and font in all controls when VCL Style is enabled!
