TscPasswordEdit

Unit

scControls.pas


Description

Password edit control with different style kinds, transparency and images for password char

Properties


Name Type Description
BorderKind TscCustomEditBorderKind Kind of border...

TscCustomEditBorderKind = (sccebFrame, sccebBottomLine, sccebNone);
BottomLineColor TColor Color for bottom line
Transparent Boolean If True then controls is transparent
PasswordKind TscPasswordKind Style of password char...

TscPasswordKind = (pkPasswordChar, pkRoundRect, pkRect, pkTriangle);
PasswordCharImages TCustomImageList Images for password char
PasswordCharImageIndex Integer Image index for password char
PasswordCharSelectedImageIndex Integer Image index for password char, which in selection area
PromptText String Prompt text value
HidePromptTextIfFocused Boolean Hide Prompt text value if control is focused
UseFontColorToStyleColor Boolean If True then color of the text will be GetStyleColor(Font.Color)

 

  Note:   

    Use TControl.StyleElements property to adjust custom colors for client and font in all controls when VCL Style is enabled!
