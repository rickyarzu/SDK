TscGPImageCollection

Unit

scGPImages.pas

Description

 

Collection of images with different drawing style. Images can be HQ scaled to any scale factor.

 

Properties

 

Name Type Description
Images TscGPImageCollectionItems Collection of TscGPImageCollectionItem

Methods

 

Name Description
Draw procedure Draw(ACanvas: TCanvas; ARect: TRect; AIndex: Integer; AScaleFactor: Double);
Draw image with special index, canvas and in specific area

 

 

TscGPImageCollectionItem


Properties

 

Name Type Description
PngImage TPngImage Png image, which will be converted to the TGPBitmap (GDI+ bitmap)

Use TscGPImageCollectionItem.LoadPngFromFile method to load png image from file.
Bitmap TBitmap Bitmap, which based from PNG image. You'll see special dialog to load PNG image. As result you'll get 32-bit bitmap with alpha channel or 24-bit bitmap, which will be converted to the TGPBitmap (GDI+ bitmap). You can choose between this property and PngImage. Convertation with bitmaps more faster, but it has a more lager size in *dfm file.
Drawing style TscImageDrawStyle TscImageDrawStyle = (idsTile, idsStretch, idsCenter,
    idsTopLeft, idsTopRight, idsBottomLeft, idsBottomRight,
    idsTopTile, idsLeftTile, idsRightTile, idsBottomTile,
    idsTopStretch, idsLeftStretch, idsRightStretch, idsBottomStretch,
    idsHorzStretchTile, idsVertStretchTile, idsHorzCenterStretch, idsVertCenterStretch,
    idsLeftCenter, idsTopCenter, idsRightCenter, idsBottomCenter);

If you set Wallpaper or CustomImage... (TscImageCollection) and WallpaperIndex, CusotmImageIndex... property in controls then TscGPImageCollection will automatically draw image with control scale factor.
TileOffsetX Integer X offset for tile draw style
TileOffsetY Integer Y offset for tile draw style
Options TscGPBitmapOptions Options, which define how image will be draw with defined margins
Description String Description to see info about image in big list
VirtualWidth, VirtualHeight Integer Virtual sizes of image for 1x scale. You can add image 100x100 for example, set virtual sizes as 50x50. Then in system with 100% DPI  you can see image with 50x50 size, in system with 175% DPI you can see image with 75x75 size,... 200% - 100x100. This gives you opportunity to get always HQ scaled image.
Proportional Boolean Use this property to keep proportional of image in some drawing styles
ContentLeftMargin
ContentRightMargin
ContentTopMargin
ContentBottomMargin		 Integer These values define area for content in the control. For example, if you use custom image for TscButton control then it will uses this values to adjust text and image position.
ScaledBooleanImage will be scaled with AScaleFactor in Draw method

Properties

 

Name Type Description
LeftMargin

BottomMargin		 Integer Margins, which defined corners, border and client part in the bitmap.

Examples:
Fixed size:
(LeftMargin = 0) and (RightMargin = 0) an (TopMargin = 0) and (BottomMargin = 0)

Fixed height:
(LeftMargin > 0) and (RightMargin > 0) and (TopMargin = 0) and (BottomMargin = 0)

Fixed width:
(LeftMargin = 0) and (RightMargin = 0) and (TopMargin > 0) and (BottomMargin > 0)

Full resize:

LeftMargin > 0) and (RightMargin > 0) and (TopMargin > 0) and (BottomMargin > 0)

StretchBorder		 Boolean Stretch or Tile drawing of client part and border
