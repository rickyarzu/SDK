TscStyleManager

Unit

scStyleManager.pas

Description

Component to adjust styled menus, styled common dialogs (Seattle+ version) and scale options. 

Styled menus works since XE2 version with this component!

Also scStyleManager unit improves some system style hooks for common dialogs (drawing and scaling (High-DPI) support).

Properties

 

Name Type Description
MenuAlphaBlendValue Byte AlphaBlend value for menus. If value < 255 then alphablend for menus is enabled
MenuImages TscImageCollection Images for wallpaper and backgrounds for menus
MenuWallpaperIndex Integer Index of image for wallpaper of menu
MenuBackgroundIndex Integer Index of image for custom background of menu
MenuBackgroundOverContentIndex Integer Index of image, which will draw over all items of menu
MenuHeadersSupport Boolean If True then you can adjust headers for styled menus.

To make some item as header to these steps:
1. Set first and last characters of TMenuItem.Caption = '-'
2. Set TMenuItem.Enabled = False
MenuHookEnabledBooleanEnabled / Disable styles for menus (for XE2-XE5) versions
For XE6 - 10 Seattle use standard TStyleManager.SystemHooks property
ScaleStylesBooleanIf True then all controls will scale some style elements for hi-dpi systems.Main problem that VCL styles not include hi-res graphics. ("Enable HighDPI" option available since 10 Seattle).
ScaleThemesBooleanIf True then all controls will scale some theme elements for hi-dpi systems.For example if you moved form to other monitor, with other dpi then checkboxes and radioboxes will be with size for current DPI. Main problem of VCL that it not properly supports change of monitors with different DPI.("Enable HighDPI" option available since 10 Seattle).
ScaleFormBorderBooleanIf True then border of styled form will be scaled to any system scale factor (125%, 150%-200% and more). All VCL style elements will be scaled automatically + buttons and tabs in caption (see TscStyledForm component).
StyleArrowTypeTscStyleArrowType TscStyleArrowType = (scsatDefault, scsatModern);

Defines what kind of arrows in some controls and menus will be used. VCL Styles have a different kind of arrows (triangles or arrows with lines (modern)).
MenuSelectionStyleTscMenuSelectionStyle
TscMenuSelectionStyle = (scmssStyled, scmssColor);

Defines what kind of selection will be used in menus (scmssStyled - use selection bitmap from style,  scmssColor - use selection color from style).  scmssColorn looks good with GP (GDI+) controls. 
RTLModeBooleanEnable BidiMode = bdRightToLeft for dialog forms
