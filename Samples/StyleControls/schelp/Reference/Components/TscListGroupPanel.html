TscListGroupPanel

Unit

scExtControls.pas

Description

Panel control, which looks as listbox group

Properties

Name Type Description
Caption String Text in header
HeaderHeight Integer Height of the header area
HeaderAutoColor Integer If True then color for  header text will be calculated automatically from current theme and style
HeaderMargin Integer Margin for header text
HeaderFont TFont Font for header text
GroupColor TColor Color of the group (client) area
GroupColorAlpha Byte Alpha value for group color
RowCount Integer Count of virtual rows in group
RowLineMargin Integer Margin for row line
TransparentBackground Boolean if True then header area will be transparent
RowHeight Integer Height of the row. (if value = 0 then row height will be automatic)
AutoSize Boolean If True and RowHeight <> 0 then height will be as RowHeight * RowCount + HeaderHeight
StorePaintBuffer Boolean If True then transparent child controls from StyleControls VCL will use special paint buffer to get parent image. This solution really increase performance!

 

Note:

    Use TControl.StyleElements property to adjust custom colors for client and font in all controls when VCL Style is enabled!
