TscGPDial

Unit

scGPMeters.pas

Description

Rounded dial control

Properties

 

Name Type Description
MinValue Double Min value
MaxValue Double Max value
Value Double Current value
FormatStrValue String Float format to display value
FormatStrLabel String Float format to display tick labels
AutoSizeFont Boolean If True then font height will be automatic else control uses Font.Size property
FillColor TColor Fill color for control shape (clNone - fill is missing)
FillColorAlpha Byte Alpha value of the fill color for control shape
FillStyle TscGPShapeFillStyle TscGPShapeFillStyle = (scgpsfColor, scgpsfGradient);

scgpsfColor - solid color
scgpsfGradient - auto gradient from FillColor
FillGradientAngle Integer Angle of fill gradient
FrameColor TColor Color of the frame (clNone - frame is invisible)
FrameWidth Integer Width of the frame
ScaleSteps Integer Count of the scale steps
ScaleSubSteps Integer Count of the scale steps between big ticks (define small ticks)
ScaleSections TscScaleSections Collection of sections on scale arc. You can set color, alpha value, StartValue and EndValue for each section
ScaleDivider Double Divider value of tick numbers (If Value = 3000 and Divider = 1000 then you get tick with "3" label)
TicksColor TColor Color of the scale ticks
TicksColorAlpha Byte Alpha value for color of the scale ticks
TicksWidth Integer Width of the tick on scale
TicksSmallWidth Integer Width for small ticks
ShowScaleTicks Boolean If True then control shows ticks on the scale
ShowValue Boolean If True then control shows value 
ShowScaleLabels Boolean If True then control shows labels on the scale
CanFocused Boolean Enable focus on control
KnobColor TColor Color of the knob
KnobColorAlpha Byte Alpha value for knob color
ValueTextColor TColor Color of the value text
KnobShadow Boolean if True then knob has inside shadow effect
ImageCollection TscCustomImageCollection Image collection component
ImageIndex Integer Index of image from image collection
ShowImage Boolean Show / Hide image
ShowProgress Boolean Show progress line (visible only when visible section or ticks)
ProgressColor TColor Color of the progresline
ProgressColorAlpha Byte Alpha value for progress color
DrawTextMode TscDrawTextMode TscDrawTextMode = (scdtmGDI, scdtmGDIPlus);

scdtmGDI - draw text with GDI methods
scdtmGDIPlus - draw text with GDI+ methods

Use scdtmGDIPlus value on the Fluent UI background (see TscStyledForm component)

Events

 

Name Type Description
OnChange TNotifyEvent Event when value changed
OnLastChange TNotifyEvent Event when value changed and you release thumb capture
