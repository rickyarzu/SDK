Description

Edit control with vector shape, background and buttons

Properties

 

Name Type Description
LeftButton TscGPEditButton Adjust left button
RightButton TscGPEditButton Adjust right button
Transparent Boolean If True then client part of edit is transparent
Prompt text String Hint text for edit
HidePromptTextIfFocused Boolean Hide Prompt text value if control is focused
Options TscGPEditOptions Options for edit shape and background
ContentMarginLeft,
ContentMarginTop,
ContentMarginRight,
ContentMarginBottom		 Integer Margins for edit client area
CustomDrawBooleanSet it True to prevent drawing any default content in non-focused control with assigned OnDrawBackground event 
 

Events

 

Name Type Description
OnLeftButtonClick TNotifyEvent Click event for left button
OnRightButtonClick TNotifyEvent Click event for right button
OnDrawBackgroundTscPaintControlEventprocedure(ACanvas: TCanvas; ARect: TRect;

Note, that ARect - is a client (text) area of the editor
