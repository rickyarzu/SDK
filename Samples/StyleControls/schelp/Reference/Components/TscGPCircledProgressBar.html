TscGPCircledProgressBar

Unit

scGPControls.pas

Description

Circled progress bar control with with vector graphic

Properties

 

Name Type Description
MinValue Integer Min value
MaxValue Integer Max value
Value Integer Current value
ProgressFont TFont Font for percents
Caption String Caption, which can be used as description of current process
Font TFont Font for Caption property
FrameSize Integer Size (width) of rouned frame line
FrameColor TColor Color of rounded frame
FrameAlpha Byte Alpha value for rounded frame
ProgressColor TColor Color of progress line
ProgressAlpha Byte Alpha value for rounded progress line
ShowCaption Boolean Show / Hide Caption
ShowProgressText Boolean Show / Hide percents info

Use OnGetProgressText event to set your text for progress value!
AnimationMode Boolean Enable animation mode in control
Active Boolean Activate animation in control
AnimationAcceleration Boolean Enabled acceleration type of animation
ImageCollection TscCustomImageCollection Image collection component
ImageIndex Integer Index of image from image collection
ShowImage Boolean Show / Hide image
DrawTextMode TscDrawTextMode TscDrawTextMode = (scdtmGDI, scdtmGDIPlus);

scdtmGDI - draw text with GDI methods
scdtmGDIPlus - draw text with GDI+ methods

Use scdtmGDIPlus value on the Fluent UI background (see TscStyledForm component)
