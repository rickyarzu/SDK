OnCloseUp TscButtonCloseUpEvent  TscButtonCloseUpEvent = procedure(Sender: TObject; AAcceptChanges: Boolean) of object;

Use this event after closing of dropdown form:

if AAcceptChanges then
    scButton1.Caption :=MyDropDownForm.scListView1.Selected.Caption;

See also TscStyledForm

Methods

 

Name Description
CloseUp Use this method on dropdown form to redraw button after closing of dropdown form...

procedure TDropDownForm.scStyledForm1CloseUp(ADropDownForm: TCustomForm;
  ADropDownControl: TControl; AAcceptChanges: Boolean);
begin
  if ADropDownControl <> nil then
  begin
    if ADropDownControl is TscButton then
      TscButton(ADropDownControl).CloseUp(AAcceptChanges);
  end;
end;

Note:

    Use TControl.StyleElements property to adjust custom colors for client and font in all controls when VCL Style is enabled!
