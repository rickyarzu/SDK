TscGPPanel

Unit

scGPControls.pas

Description

Panel control with vector graphic for frame shape

Properties

Name Type Description
Caption String Text on control
ShowCaption Boolean Show text on panel
CaptionGlowEffect TscGlowEffect Adjust glow effect for caption
Alignment TAlignment Alignment for caption
FrameColor TColor Color of frame
FrameColorAlpha Byte Alpha value for frame
FrameRadius Integer Radius of the frame (0 value is rectangle frame)
FrameSides TscGPPanelFrameSides TscGPPanelFrameSide = (gppfsLeft, gppfsTop, gppfsRight, gppfsBottom);
TscGPPanelFrameSides = set of TscGPPanelFrameSide;

You can disable any frame side with this property
FillColor TColor Color for background
FillColor2 TColor Second color for background with gradient style.
If value is clNone then this color will be ignored and gradient filling will uses only one color
FillColorAlpha Byte Alpha value for color of background
ScaleFrameWidth Boolean If True then control will scale frame width (depends on the system DPI)
FillStyle TscGPShapeFillStyle TscGPShapeFillStyle = (scgpsfColor, scgpsfGradient);

Filling style for panel.
 scgpsfColor - one color
 scgpsfGradient - control automatically create light and dark colors for gradient from FillColor value.
FillGradientAngle Integer Angle for gradient (90, -90, 45, -125, ....)
FillGradientBeginAlpha Byte Alpha value for color in the begin of gradient
FillGradientEndAlpha Byte Alpha value for color in the end of gradient
FillGradientBeginColorOffset Byte Lightness offset in percents for first color in gradient
FillGradientEndColorOffset Byte Darkness offset in percents for second color in gradient
StorePaintBuffer Boolean If True then transparent child controls from StyleControls VCL will use special paint buffer to get parent image. This solution really increase performance!
Wallpapers TscGPImageCollection Collection of images for wallpaper, which draws on the bottom 
WallpaperIndex Integer Index of picture for wallpaper
CustomImages TscGPImageCollection Collection of images for additional image, which draws on the top
WallpaperIndex Integer Index of image for additional background
Sizeable Boolean If True the control can be resizable with mouse
If Align = alClient then parent control or form will be resizable with mouse
WordWrap Boolean If True then caption will be with word wrap.
BackgroundStyle TscGPPanelBGStyle TscGPPanelBGStyle = (gppbsColor, gppbsFormBackground);

Background style for panel (if TransparentBackground is False):

gppbsColor - color (TscGPPanel.Color property)
gppbsFormBackground - background as for form (if VCL Style enabled then it will be bitmap background as defined in style).
FluentUIOpacity Boolean Set it to True to get opacity area of the form with Fluent UI (see TscStyledForm component)
DrawTextMode TscDrawTextMode TscDrawTextMode = (scdtmGDI, scdtmGDIPlus);

scdtmGDI - draw text with GDI methods
scdtmGDIPlus - draw text with GDI+ methods

Use scdtmGDIPlus value on the Fluent UI background (see TscStyledForm component)
ContentMarginLeft,
ContentMarginTop,
ContentMarginRight,
ContentMarginBottom		 Integer Margins for caption and client area
