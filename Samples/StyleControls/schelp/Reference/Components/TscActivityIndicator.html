TscActivityIndicator

Unit

scModernControls.pas

Description

Control to show animated activity indicator

Properties

 

Name Type Description
Active Boolean If True then animation is active
IndicatorColor TColor Color value for indicator. For example, if you set Color = clBtnText then this value automatically will be from style. Also you can set any custom color, for example, clRed.
KindTscActivityIndicatorKindTscActivityIndicatorKind = (scaikPoints, scaikCircle, scaikSectors);

Kind of indicator

Note

You can change width and height of the control from 32 to 64 values. Optimal size is 32 or 64. 

For Kind = scaikCircle size is 64x64 only!
