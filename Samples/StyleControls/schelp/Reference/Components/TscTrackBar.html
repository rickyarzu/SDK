TscTrackBar

Unit

scControls.pas

Description

Trackbar control with different styles

Properties

 

Name Type Description
MinValue Integer Min value
MaxValue Integer Max value
Value Integer Current value
MouseSupport Boolean If True then you can change value with mouse
JumpWhenClick Boolean If True then you can change value by click
ShowFocusRect Boolean Show focus rect
CanFocused Boolean Enabled focus state for button
CustomImages TscImageCollection TscImageCollection component. Bitmap property must be defined in all items, which you will use for custom style
TrackOptionsTscTrackBarTrackOptionsOptions to adjust style for track area (colors or custom image indexes for different states, size of track area and more)
ThumbOptionsTscTrackBarThumbOptionsOptions to adjust style for thumb  (colors or custom image index for different states, width and height of the thumb and more)
VerticalBooleanVertical kind
MouseWheelSupportBooleanEnable mouse wheel to change value
MouseWheelOppositeBooleanIf True then increasing and decreasing of the value will be opposite from mouse wheel delta

 

Events

 

Name Type Description
OnChange TNotifyEvent Event when value changed
OnLastChangeTNotifyEventEvent when value changed and you release thumb capture
 
 Note:

    Use TControl.StyleElements property to adjust custom colors for client and font in all controls when VCL Style is enabled!
