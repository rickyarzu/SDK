TscPageViewer

Unit

scModernControls.pas

Description

Simple page control only to switch between pages. (analogue of TNoteBook).

Properties

  

Name Type Description
Pages TscPageViewerItems Collection of TscPageViewerItem.
Use TscPageViewerItem.Page property to get page for item.
Component use this collection to get index of page. Also you can add, delete or reorder items (pages) with collection editor.
PageIndex Integer Index of page in collection
ActivePage TscPageViewerPage TscPageViewerPage = class(TscScrollBox)

Events

  

Name Type Description
OnChangePage TNotifyEvent Event when page was changed
