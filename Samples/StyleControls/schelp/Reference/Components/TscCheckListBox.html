TscListBox = class(TCheckListBox)

Description

CheckListBox control with additional features

Properties


Name Type Description
Images TCustomImageList Images for item
ImageIndex Integer Global image index for item. Default value is -1 (in this case Index for image is Index of item)
WordBreak Boolean Draw text in item with word break
TitleDivider String Devider string in items  text to detect title string. For example, if TitleDivider = @ then to draw title in item you must add: Title1@Item1
LineColor TColor Color of the lines
ShowLines Boolean Show lines between items
SelectionStyle TscSelectionStyle Kind of selection:

TscSelectionStyle = (scstStyled, scstColor);
scstStyled - draw selection as bitmap from theme or VCL Style
scstColor - draw selection as highlight color from theme or VCL Style
ShowFocusRect Boolean Show focus rect for item
SelectionColorTColorCustom selection color for control (value must be <> clNone)
SelectionTextColorTColorColor of text for custom selection color
