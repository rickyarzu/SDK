TscGPToolGroupPanel

Unit

scGPPagers.pas

Description

Ribbon group analogue with vector dialog button

Properties


Name Type Description
ShowDialogButton Boolean Show special button in bottom right corner to call some dialog
CaptionHeight Integer Height of the caption
StorePaintBuffer Boolean If True then transparent child controls from StyleControls VCL will use special paint buffer to get parent image. This solution really increase performance!
DrawTextMode TscDrawTextMode TscDrawTextMode = (scdtmGDI, scdtmGDIPlus);

scdtmGDI - draw text with GDI methods
scdtmGDIPlus - draw text with GDI+ methods

Use scdtmGDIPlus value on the Fluent UI background (see TscStyledForm component)
 

Events

 

Name Type Description
OnDialogButtonClick TNotifyEvent Click event for dialog button
