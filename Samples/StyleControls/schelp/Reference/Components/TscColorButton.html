Button to select color

See TscButton properties also

Name Type Description

AutoCaption String Caption for AutoColor item
ShowMoreColor Boolean Show item to call color dialog
ShowCustomColors Boolean Show items for custom colors from color dialog
MoreCaption String Caption for MoreColor item
ColorValue TColor Current color value
ColorDialog TColorDialog TColorDialog component
MenuBGStyle TscGalleryMenuBGStyle Style for popup grid background
MenuFont TFont Font for popup color grid
ColorGridType TscColorGridType Type of color's grid...

scgsExtended - rich color grid
scgsSimple - simple color grid
CustomColorsTscCustomColorButtonValues = array[1..8] of TColor;Values of custom colors

Name Type Description
