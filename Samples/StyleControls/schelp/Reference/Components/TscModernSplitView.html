TscModernSplitView

Unit

scModernControls.pas

TscModernSplitView = class(TscSplitView)

Description

SplitView control to create Windows 10 Modern Application. 

It can interact with TscButton, TscPageViewer and TscSplitView controls with special properties amd methods.

Properties

 

Name Type Description
PageViewer TscPageViewer Page control to show pages. Control automatically save browsing history.
BottomSplitView TscSplitView Bottom split panel. Control automatically can closed it.


Events

 

Name Type Description
OnShowBackButton TNotifyEvent Use this event to show or enable your button with "back" functionality
OnHideBackButton TNotifyEvent Use this event to hide or disable your button with "back" functionality


Methods

 

Name Description
OpenPage procedure OpenPage(APageIndex: Integer; AButton: TscButton);
Call this method to open page in PageViewer control and link some button to the page.
Control automatically will  show and hide PageViewer control.
This action will be added to the browsing history.

Put TscButton controls to the TscModernSplitView and adjust style and color of them.
Use TscButton.OnClick event to call OpenPage method.
Set TscButton.GroupIndex property to get one group for buttons, which can open pages.
Back procedure Back;
Call this method to execure previous action from browsing history.
CloseAllprocedure CloseAll;
Call this method to close control, PageView control and BottomSplitView control.
