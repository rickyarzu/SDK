TscListView

Unit

scControls.pas

TscListView =  class(TCustomListView)

Description

ListView control with additional features

Properties


Name Type Description
SelectionStyle TscSelectionStyle Kind of selection:

TscSelectionStyle = (scstStyled, scstColor);
scstStyled - draw selection as bitmap from theme or VCL Style
scstColor - draw selection as highlight color from theme or VCL Style
ShowFocusRect Boolean Show focus rect for item
AlternateRow Boolean If True then control has alternate row colors in report mode
DefaultDraw Boolean If True then control not override system painting when VCL Style not enabled
SelectionColor TColor Custom selection color for control (value must be <> clNone)
SelectionTextColor TColor Color of text for custom selection color
ExtendedColumnDrawBooleanIf True then control will draws progressbar or checkbox in columns automatically.

Set Column.Tag = 1 to draw progressbar (text must be '0'..'100')
Set Column.Tag = 2 to draw checkbox (text must be '1' for checked state)

 

Note:

    Use TControl.StyleElements property to adjust custom colors for client and font in all controls when VCL Style is enabled!
