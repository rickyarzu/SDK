TscGPListBox

Unit

scGPExtControls.pas

Description

ListBox control with vector graphic for scrollbars, backgroud, selection, headers and more

Properties

  

Name Type Description
Options TscGPScrollingControlOptions Options for frame and background (see "Scale..." properties to on / off scale of some parameters)
ScrollBarOptions TscGPControlScrollBarOptions Options for scrollbars
SelectionOptions TscGPSelectionOptions Options for selection
HeaderOptions TscGPHeaderOptions Options for header
CheckBoxOptions TscGPListCheckBoxOptions Options for checkboxes (see "Scale..." properties to on / off scale of some parameters)
Items TscGPListBoxItems Collection of TscGPListBoxItem
ItemIndex Integer Index of active item
ShowItemTitles Boolean Show item's title
ItemHeight Integer Height of item
HeaderHeight Integer Height of items with Header = True
ShowLines Boolean Show divider lines between items
LineColor TColor Color of lines
LineColorAlpha Byte Alpha value for lines
MouseMoveChangeIndex Boolean If True then ItemIndex will change with mouse
HeaderFont TFont Font for header items
Color TColor Color value for scbgsColor background style
Images TCustomImageList Images for items
DetailFont TFont Font for detail in item
ShowItemDetails Boolean If true then item have area for detail on bottom
ItemMargin Integer Margin between border and item's content
ItemSpacing Integer Spacing between image and text in item
Filter String Filter string for listbox
IndentMargin Integer Margin value for indent (offset = item.indent * IndentMargin)
ItemWordWrap Boolean Enable word wrap for item caption
ItemShowEllipsis Boolean If True then long item caption can has ellipsis
Wallpapers TscGPImageCollection Collection of images for wallpaper, which draws on the bottom 
WallpaperIndex Integer Index of picture for wallpaper
CustomImages TscGPImageCollection Collection of images for additional image, which draws on the top
WallpaperIndex Integer Index of image for additional background
DetailPosition  TscGPListBoxDetailPosition TscGPListBoxDetailPosition = (scglbdBottom, sclgbdRight);

Position of the detail text in item (bottom or right)
DetailWordWrap Boolean Enable word wrap for detail text
DrawTextMode TscDrawTextMode TscDrawTextMode = (scdtmGDI, scdtmGDIPlus);

scdtmGDI - draw text with GDI methods
scdtmGDIPlus - draw text with GDI+ methods

Use scdtmGDIPlus value on the Fluent UI background (see TscStyledForm component)

Events

  

Name Type Description

OnItemCheckClick TNotifyEvent Event on item checkbox click (use ItemIndex property to get active item)
OnItemButtonClick TNotifyEvent Event on item button click (use ItemIndex property to get active item)
