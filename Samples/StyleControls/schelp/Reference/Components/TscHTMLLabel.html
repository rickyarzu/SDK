TscHtmlLabel

Unit

scHtmlControls.pas

Description

Label control, which has support of html tags in caption

Properties

 

Name Type Description
Caption
 String Caption can has html tags, such as:
<a>, <a href="somelink"></a>
<font size=xxx, color=yyy, bgcolor=zzz></font>
<br>,
<b></b>,<i></i>,<s></s>,<u></u>

Color value  can be Integer value, HEX value and string value (orange,blue,red,green,...).
Expressions, such as size=xxx, color=xxx, bgcolor=xxx, href=, must be without spaces.

Example:
<font color=orange><b>TscHtmlLabel:</b></font><br>
<font color=red>Please</font>, visit <b>our website</b><br>at address: <a href="http://www.almdev.com">www.almdev.com</a><br>or <a>About</a> page in <font color=white bgcolor=red><i>this demo</i></font> !
AutoSizeWidthBooleanAdjust width automatically
AutoSizeHeightBooleanAdjust height automatically
LinkFontNormalTFontFont for link  in normal state
LinkFontHoverTFontFont for link in mouse over state
MaxWidthIntegerMax width of the control
TextAlignmentTAlignmentAlignment of html text in control
AutoExecuteURLBooleanif True then control will automatically call  ShellExecute method with href value
