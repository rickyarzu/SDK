StyleControls Overview

StyleControls VCL is a powerfull package of components, which uses Classic drawing, system Themes, GDI+ and VCL Styles. This package contains the unique solutions to extend standard VCL controls and also has many unique, advanced controls to create Windows 10 Modern UI Applications (UWP design) with Fluent UI background.  Also with this package you can really improve applying and using of VCL Styles in your application.

 

StyleControls is intended for use with:

  • Delphi XE2-10.4 Sydney
  • C++Builder XE2-10.4 Sydney
